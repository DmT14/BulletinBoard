# BulletinBoard — электронная доска объявлений
## Описание проекта
Проект выполнялся в рамках курсовой работы. Цель проекта — разработка веб-приложения доски объявлений. Реализация велась на языках HTML, CSS, JavaScript и PHP, используемая СУБД — phpMyAdmin, сборка веб-сервера — XAMPP (локально). Используемый редактор — Visual Studio Code. Оценка по результатам защиты — «отлично».

## Скриншоты приложения
Далее приведены некоторые скриншоты работы веб-приложения.

При переходе на сайт открывается Главная страница. Числом «1» обозначена форма авторизации. Числом «2» обозначено меню выбора категорий объявлений: в каждой категории выдаются объявления по тематике, соответствующей выбранной категории.
